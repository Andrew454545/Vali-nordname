# Vali-nordname

Your fork of Vali for adding a **reject locations with road name** option.

- **Repo:** https://github.com/Andrew454545/Vali-nordname
- **Original:** https://github.com/slashP/Vali

## Working in this project

1. Open this folder in Cursor: **File → Open Folder** → choose `Vali-nordname`.
2. Install **.NET 8 SDK** if you haven’t: https://dotnet.microsoft.com/en-us/download/dotnet/8.0
3. Build/run from terminal: `dotnet build` or `dotnet run` (see readme.md for Vali commands).

## Adding “no road names”

See the guide in the main Geonections folder: **VALI-FORK-ADD-NO-ROAD-NAME.md** (in the parent directory). It describes adding `RejectRoadName` to the map definition and a runtime check using the same Google panorama metadata as Various Map Generator.
